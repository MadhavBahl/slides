8 ways to find open source projects/issues

Check out the full video on YouTube by #TheLeanProgrammer

1. Issue Hub

A website which helps you find projects/issues based on your required language or issue labels.

http://issuehub.io/

2. Code Triage

Another useful tool for searching issues and projects

https://www.codetriage.com/

3. Up for grabs

List of projects with issues that can be resolved by beginners

https://up-for-grabs.net/

4. Contrib

Good list of issues and repositories

https://gauger.io/contrib/

5. First Timers Only

A list of issues that are labelled "first-timers-only"

https://www.firsttimersonly.com/

6. Awesome First Timers

A list of awesome beginners-friendly projects

https://github.com/MunGell/awesome-for-beginners

7. Pull Request Roulette

List of PRs that needs to be reviewed

http://www.pullrequestroulette.com/

8. GitHub Search

Yes, let's search it ourselves ;)

http://github.com/search
